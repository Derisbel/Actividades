Actividad 1: Introducción a CSS

1. ¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación de un documento escrito en HTML o XML.

2. ¿Cuál es la función principal de CSS?

La función principal de CSS es separar el contenido de un documento de su presentación, permitiendo así un diseño más flexible y controlado.

3. ¿Cómo se puede incluir CSS en un documento HTML?

CSS se puede incluir en un documento HTML de tres maneras:

  • En línea: mediante el atributo style en un elemento.
  • Interno: dentro de una etiqueta "style" en la sección "head".
  • Externo: enlazando un archivo CSS con una etiqueta .

    • 4. ¿Qué son los selectores en CSS?

    Los selectores en CSS son patrones que se utilizan para seleccionar los elementos que se quieren estilizar.

    5. ¿Qué son las propiedades y valores en CSS?

    Las propiedades en CSS son características que se pueden aplicar a los selectores, mientras que los valores son los ajustes específicos de esas propiedades.

    6. ¿Cómo se aplica el modelo de caja en CSS?

    El modelo de caja en CSS describe la estructura de un elemento, que incluye márgenes, bordes, relleno y el contenido en sí.

    7. ¿Qué es el "cascading" en CSS?

    El "cascading" en CSS se refiere a la forma en que se aplican las reglas de estilo, priorizando ciertas reglas según su especificidad y origen.

    8. ¿Qué son las media queries?

    Las media queries son técnicas que permiten aplicar estilos diferentes según las características del dispositivo, como el tamaño de la pantalla.

    9. ¿Cuáles son algunos estilos comunes que se pueden aplicar?

    Algunos estilos comunes que se pueden aplicar incluyen colores de texto, tamaños de fuente, márgenes, padding, bordes, y fondos.

  • Colores de texto
  • Tamaños de fuente
  • Márgenes
  • Rellenos (padding)
  • Bordes
  • Fondos

    • 10. ¿Cómo se puede usar CSS para crear un diseño responsivo?

    CSS se puede usar para crear un diseño responsivo mediante el uso de media queries y unidades flexibles como porcentajes y ems, adaptando así el diseño a diferentes tamaños de pantalla.
