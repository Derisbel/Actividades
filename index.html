1.¿Que es un sistema operativo?

son los programas encargados de administrar y gestionar de manera ç eficiente todos los recursos de un ordenador y otros dispositivos

2.Componentes de un Sistema Operativo

Núcleo

En informática, un núcleo o kernel (de la raíz germánica Kern, 'núcleo', 'hueso') es un software que constituye una parte fundamental del sistema operativo, y se define como la parte que se ejecuta en modo privilegiado (conocido también como modo núcleo).

Controladores de dispositivos

Un controlador de dispositivo o manejador de dispositivo (en inglés: device driver, o simplemente driver)​ es un programa informático que permite al sistema operativo interactuar con un periférico, haciendo una abstracción del hardware y proporcionando una interfaz (posiblemente estandarizada) para utilizar el dispositivo.7​

Gestión de procesos

Un proceso es simplemente, un programa en ejecución que necesita recursos para realizar su tarea: tiempo de CPU, memoria, archivos y dispositivos de E/S. El SO es el responsable de lo siguiente:

  • Crear y destruir procesos.
  • Parar y reanudar procesos.
  • Ofrecer mecanismos para que los procesos puedan comunicarse y se sincronicen.

    • Gestión de la memoria

    La gestión de memoria o administración de memoria es el acto de gestionar la memoria de un dispositivo informático. El proceso de asignación de memoria a los programas que la solicitan.

    Gestión del almacenamiento secundario

    Un sistema de almacenamiento secundario es necesario, ya que la memoria principal (almacenamiento primario) es volátil y además muy pequeña para almacenar todos los programas y datos. También es necesario mantener los datos que no convenga mantener en la memoria principal. El SO se encarga de:

  • Planificar los discos.
  • Gestionar el espacio libre.
  • Asignar el almacenamiento.
  • Verificar que los datos se guarden en orden.

    • Sistema de entrada y salida

    Consiste en un sistema de almacenamiento temporal (caché), una interfaz de manejadores de dispositivos y otra para dispositivos concretos. El sistema operativo debe gestionar el almacenamiento temporal de E/S y servir las interrupciones de los dispositivos de E/S.

    Sistema de archivos

    Un sistema de archivos o sistema de ficheros, (file system: fs), en informática, es un elemento que controla cómo se almacenan y recuperan los datos. Sin un sistema de archivos, los datos colocados en un medio de almacenamiento serían un gran cuerpo de datos sin manera de saber dónde termina un dato y comienza el siguiente. Es el encargado de administrar y facilitar el uso de las memorias periféricas o internas, ya sean secundarias o terciarias.​

    Sistemas de protección

    Mecanismo que controla el acceso de los programas o los usuarios a los recursos del sistema. El SO se encarga de:

  • Distinguir entre uso autorizado y no autorizado.
  • Especificar los controles de seguridad a realizar.
  • Forzar el uso de estos mecanismos de protección.

    • Sistema de comunicaciones

    Para mantener las comunicaciones con otros sistemas es necesario poder controlar el envío y recepción de información a través de las interfaces de red. También hay que crear y mantener puntos de comunicación que sirvan a las aplicaciones para enviar y recibir información, y crear y mantener conexiones virtuales entre aplicaciones que están ejecutándose localmente y otras que lo hacen remotamente.

    Programas de sistema

    Son aplicaciones de utilidad que se suministran con el SO pero no forman parte de él. Ofrecen un entorno útil para el desarrollo y ejecución de programas, siendo algunas de las tareas que realizan:


  • Manipulación y modificación de archivos.
  • Información del estado del sistema.
  • Soporte a lenguajes de programación.
  • Comunicaciones.

    • Gestor de recursos

    Como gestor de recursos, el sistema operativo administra:

  • La unidad central de procesamiento (donde está alojado el microprocesador).
  • Los dispositivos de entrada y salida.
  • La memoria principal (o de acceso directo).
  • Los discos (o memoria secundaria).
  • Los procesos (o programas en ejecución).

    • Interfaz de usuario

    La interfaz de usuario, IU (del inglés User Interface, UI), es el medio que permite la comunicación entre un usuario y una máquina, equipo, computadora o dispositivo, y comprende todos los puntos de contacto entre el usuario y el equipo

    3.Tareas del Sistema Operativo

    Las tareas del sistema operativo incluyen la gestión de recursos del hardware (como la CPU, la memoria y los dispositivos de entrada/salida), la administración de archivos y directorios, la gestión de procesos (ejecución de programas), la gestión de la seguridad y control de acceso, así como la interfaz de usuario para interactuar con el sistema.

    4.Funciones de un Sistema Operativo

    Las funciones principales de un sistema operativo son gestionar recursos del hardware, proporcionar una interfaz para que los usuarios y las aplicaciones interactúen con el sistema, facilitar la ejecución de programas (gestión de procesos), administrar archivos y directorios, controlar la seguridad y acceso a recursos, y gestionar la memoria y los dispositivos.

    5.Características de un Sistema Operativo

    Las características típicas de un sistema operativo incluyen la multitarea (capacidad para ejecutar varios programas simultáneamente), la multiprocesamiento (gestión eficiente de múltiples núcleos de CPU), la multiusuario (soporte para múltiples usuarios concurrentes), la portabilidad (capacidad de funcionar en diferentes tipos de hardware), la seguridad (control de acceso a recursos), la compatibilidad con redes, y una interfaz de usuario.

    6.Tipos de Sistema Operativo

    Los tipos de sistemas operativos incluyen sistemas operativos de escritorio (como Windows, macOS, y varias distribuciones de Linux), sistemas operativos móviles (iOS, Android), sistemas embebidos (para dispositivos como routers o electrodomésticos inteligentes), sistemas de tiempo real (para aplicaciones que requieren respuestas rápidas y deterministas), y sistemas distribuidos (que coordinan múltiples computadoras para trabajar en conjunto).

    7.Sistemas Operativos Para PC

    Los sistemas operativos más comunes para PC incluyen Microsoft Windows (como Windows 10), macOS (anteriormente OS X), y diversas distribuciones de Linux (como Ubuntu, Fedora, y Debian). Estos sistemas están diseñados para gestionar eficientemente recursos como la memoria, el procesador, los dispositivos de almacenamiento y periféricos en computadoras personales.

    8.Sistemas Operativos Móviles

    En el ámbito de dispositivos móviles como smartphones y tablets, los sistemas operativos más utilizados son Android (desarrollado por Google), iOS (desarrollado por Apple) y en menor medida, otros sistemas como Windows Phone (Microsoft) y algunos derivados de Linux. Estos sistemas operativos están optimizados para pantallas táctiles, conectividad móvil y gestión de aplicaciones diseñadas específicamente para estos dispositivos.

    9.Sistema Operativo según el Ordenador y el Tipo de Aplicaciones

    Los sistemas operativos pueden variar según el tipo de computadora y las aplicaciones que se ejecuten. Por ejemplo, los sistemas operativos para servidores difieren de los sistemas para PCs o dispositivos móviles en términos de gestión de recursos y seguridad. También existen sistemas embebidos para dispositivos especializados como cajeros automáticos o sistemas de navegación.

    10.Compatibilidad de los Sistema Operativo

    La compatibilidad de un sistema operativo se refiere a su capacidad para ejecutar aplicaciones y trabajar con hardware específico. Algunos sistemas operativos son más compatibles con ciertas aplicaciones y dispositivos que otros, lo que puede afectar la elección del sistema operativo dependiendo de las necesidades del usuario o de la empresa.

    11.Cuál es la función básica de un Sistema Operativo

    La función básica de un sistema operativo es gestionar los recursos del hardware y proporcionar servicios a las aplicaciones de software. Esto incluye la gestión de memoria, el control de acceso a los recursos, la planificación del procesador, la gestión de archivos y la interfaz de usuario.

    12.Mencione tres Sistema Operativo más utilizados

    Los tres sistemas operativos más utilizados son Microsoft Windows, macOS y Linux. Cada uno tiene sus propias características y es utilizado en diferentes tipos de dispositivos y por diferentes usuarios en todo el mundo.

    13.Menciona algunas tareas que realizan los Sistemas Operativos

    Entre las tareas que realizan los sistemas operativos se incluyen la gestión de memoria (asignación y liberación de recursos), gestión del sistema de archivos (lectura, escritura y organización de archivos en el almacenamiento), gestión de procesos (control y ejecución de programas), gestión de dispositivos (control de hardware como impresoras y unidades de disco) y gestión de seguridad (control de acceso y protección de datos).

    14.Qué es un sistema operativo multitareas

    Un sistema operativo multitareas es aquel que permite ejecutar múltiples aplicaciones o procesos al mismo tiempo, compartiendo eficientemente los recursos del sistema como la CPU y la memoria entre ellos. Esto permite a los usuarios realizar varias tareas simultáneamente sin que una aplicación monopolice los recursos del sistema.

    15.Qué es un sistema operativo multiusuario

    Un sistema operativo multiusuario permite que múltiples usuarios accedan y utilicen el sistema simultáneamente. Cada usuario tiene su propio entorno de trabajo y configuración personalizada, con protección de datos y recursos para garantizar la seguridad y la privacidad.

    16.Cómo funciona un sistema operativo

    Un sistema operativo funciona coordinando las actividades del hardware y el software. Administra los recursos del sistema, proporciona interfaces para que los usuarios y las aplicaciones interactúen con el hardware, y asegura que el sistema sea seguro y eficiente.

    17.Cómo se utiliza un sistema operativo en un entorno gráfico

    En un entorno gráfico, el sistema operativo proporciona una interfaz gráfica de usuario (GUI) que permite a los usuarios interactuar con el sistema mediante iconos, ventanas y menús visuales. Los sistemas operativos modernos como Windows y macOS utilizan GUIs parafacilitar la navegación y la ejecución de aplicaciones

    18.Cuándo inicia el funcionamiento del Sistema Operativo

    El funcionamiento del sistema operativo inicia cuando se enciende o reinicia el ordenador. Este proceso se conoce como "arranque" (boot). Durante este proceso, se carga el núcleo del sistema operativo en la memoria principal (RAM) desde el disco duro u otro medio de almacenamiento, y se inicializan todos los componentes esenciales para que el sistema esté operativo

    19.Mencione las dos interfaces que forman parte del Sistema Operativo

    Las dos interfaces principales que forman parte de un sistema operativo son:

  • Interfaz de línea de comandos (CLI): Permite interactuar con el sistema operativo mediante comandos escritos en texto plano.

  • Interfaz gráfica de usuario (GUI): Proporciona una forma visual e intuitiva de interactuar con el sistema operativo mediante iconos, ventanas, menús y controles gráficos

    • 20.Qué es un proceso en entorno de los sistemas operativos

    En el contexto de los sistemas operativos, un proceso es una instancia de un programa en ejecución. Un proceso incluye el código del programa, datos asociados, registros de estado, recursos del sistema (como memoria y CPU) y un contexto de ejecución. Los sistemas operativos gestionan múltiples procesos simultáneamente utilizando técnicas como la planificación de procesos para garantizar una asignación eficiente de los recursos del sistema.

    21.Qué es un sistema operativo distribuido y centralizado

  • Sistema Operativo Centralizado: En este tipo de sistema, todas las funciones y recursos del sistema operativo son controlados y gestionados desde un solo lugar central. Los usuarios se conectan a este centro para acceder a los recursos compartidos.

  • Sistema Operativo Distribuido: En contraste, un sistema operativo distribuido distribuye el control y la gestión de los recursos entre varios nodos o dispositivos interconectados. Los recursos y servicios pueden estar distribuidos físicamente pero se presentan a los usuarios como un sistema coherente.

    • 22.Mencione los sistemas de archivos utilizados por los Sistemas Operativos

    Algunos sistemas de archivos utilizados por los sistemas operativos incluyen:

  • FAT (File Allocation Table): Utilizado por sistemas operativos como MS-DOS y versiones antiguas de Windows.

  • NTFS (New Technology File System): Utilizado por sistemas operativos modernos de Windows para ofrecer mayor seguridad y capacidad.

  • ext4 (Fourth Extended Filesystem): Utilizado por sistemas operativos basados en Linux para soportar tamaños de archivos grandes y sistemas de archivos extendidos.

  • HFS+ (Hierarchical File System Plus): Utilizado por sistemas operativos de macOS antes de macOS Catalina, que ahora utiliza APFS (Apple File System).

    • 23.Mencione otras de las características actuales de los sistemas operativos

    Algunas características actuales de los sistemas operativos incluyen:

  • Virtualización: Capacidad para ejecutar múltiples sistemas operativos o entornos aislados en un solo hardware físico.

  • Seguridad mejorada: Implementación de técnicas avanzadas de cifrado, control de acceso y detección de amenazas para proteger datos y sistemas.

  • Soporte para dispositivos móviles: Integración de funciones específicas para gestionar recursos en dispositivos móviles como smartphones y tablets.

  • Actualizaciones automáticas: Capacidad para descargar e instalar automáticamente parches de seguridad y nuevas versiones del sistema operativo para mejorar la estabilidad y funcionalidad.

    • 24.Que permite la exigencia de muchos controladores en un sistema operativo

    La exigencia de muchos controladores en un sistema operativo permite la compatibilidad y la gestión adecuada de hardware diverso. Los controladores son programas que facilitan la comunicación entre el sistema operativo y los dispositivos de hardware conectados al sistema, como impresoras, tarjetas de red, tarjetas gráficas, etc. La disponibilidad de una amplia gama de controladores asegura que los usuarios puedan utilizar hardware variado sin problemas de compatibilidad y puedan aprovechar al máximo las capacidades de sus dispositivos.
