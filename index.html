Preguntas sobre CSS

1. Definición de CSS

CSS (Cascading Style Sheets) es un lenguaje de estilos utilizado para describir la presentación de un documento escrito en HTML o XML. Permite controlar el diseño, el formato y la apariencia visual de las páginas web.

2. Funcionamiento Básico de CSS

CSS funciona mediante la asociación de estilos a elementos HTML a través de selectores. Los estilos se aplican a los elementos definidos en las reglas CSS, permitiendo cambiar colores, fuentes, márgenes, y más.

3. Incluir CSS en un archivo HTML

Existen tres formas de incluir CSS en un archivo HTML:

  • Inline: usando el atributo style en el elemento HTML.

  • Internal: dentro de una etiqueta style en la sección ,head del documento.

  • External: vinculando un archivo CSS externo usando la etiqueta ,link.

4. Definición de una regla CSS

Una regla CSS define un conjunto de estilos aplicados a uno o más selectores. Consiste en un selector y un bloque de declaración que contiene propiedades y valores.

5. Elemento de una regla CSS

Una regla CSS se compone de varios elementos:

  • Regla: Conjunto completo que incluye selector y bloque de declaración.

  • Selector: Identifica los elementos HTML a los que se aplicarán los estilos.

  • Bloque de declaración: Contiene las propiedades y valores de estilo.

  • Propiedad: Especifica el aspecto que se desea modificar (por ejemplo, color, font-size).

  • Valor: Define cómo se aplicará la propiedad (por ejemplo, red, 16px).

6. Medios CSS

Los medios CSS permiten aplicar estilos específicos según el dispositivo o las características del entorno. Se utilizan consultas de medios (@media) para adaptar el diseño a diferentes pantallas, como móviles o computadoras de escritorio.

7. Comentarios CSS

Los comentarios en CSS se utilizan para documentar el código y se escriben entre /* y */. No afectan la presentación del documento.

8. Sintaxis de la definición de las propiedades CSS

La sintaxis para definir propiedades en CSS es la siguiente:

selector {
  propiedad: valor;
}

Por ejemplo:

p {
  color: blue;
  font-size: 16px;
}
